# Green Destinations Analytics Dashboard

## Project Overview
This project analyzes employee attrition and workforce demographics for Green Destinations using an interactive Tableau dashboard. The objective is to identify key attrition drivers and provide insights to support HR decision-making.

## Key Objectives
- Analyze employee attrition rate and active workforce
- Identify department-wise and gender-wise attrition
- Study attrition trends across age groups and education fields
- Evaluate job satisfaction levels across roles

## Tools & Technologies
- Tableau Desktop (Public Edition)
- Microsoft Excel / CSV Dataset
- Data Visualization & Analytics

## Key Insights
- Overall attrition rate is **16.12%**
- Highest attrition observed in **Research & Development**
- Employees aged **25–34** show maximum attrition
- Life Sciences education background has the highest attrition count
- Job satisfaction varies significantly by role
